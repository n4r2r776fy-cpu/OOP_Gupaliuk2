###  Контрольні питання (коротко)

1. **Що таке наслідування?**  
   Механізм, який дозволяє створювати нові класи на основі вже існуючих, успадковуючи їх властивості та методи.

2. **Різниця між virtual та abstract:**  
   `virtual` — має базову реалізацію, яку можна перевизначити;  
   `abstract` — не має реалізації, вимагає перевизначення у похідних класах.

3. **Ключове слово base:**  
   Використовується для виклику конструктора або методів базового класу.

4. **Поліморфізм часу виконання:**  
   Виклик різних реалізацій методу через один інтерфейс (`virtual`/`override`).

5. **Композиція vs наслідування:**  
   Композиція — “має” (об’єкти як поля),  
   Наслідування — “є” (клас є різновидом базового).
